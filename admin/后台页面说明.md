# 后台管理系统页面说明

## 一、页面列表

### 1. 仪表盘 (Dashboard)
- **路径**：`/admin/dashboard`
- **功能**：显示系统概览数据
  - 保司数量
  - 产品数量
  - 方案数量
  - 已配置责任数量

### 2. 配置管理

#### 2.1 保司管理 (InsuranceCompanyList)
- **路径**：`/admin/insurance-companies`
- **功能**：管理保险公司的基本信息
  - 查看保司列表
  - 新增/编辑/删除保司
  - 搜索和筛选

#### 2.2 责任管理 (LiabilityList)
- **路径**：`/admin/liabilities`
- **功能**：管理保险责任
  - 查看责任列表
  - 新增/编辑/删除责任
  - 按保司筛选

#### 2.3 产品管理 (ProductList)
- **路径**：`/admin/products`
- **功能**：管理保险产品
  - 查看产品列表
  - 新增/编辑/删除产品
  - 关联保司和产品信息

#### 2.4 方案管理 (PlanList)
- **路径**：`/admin/plans`
- **功能**：管理产品方案
  - 查看方案列表
  - 新增/编辑/删除方案
  - 配置方案责任（跳转到方案责任配置页面）

#### 2.5 方案责任配置 (PlanLiabilityConfig)
- **路径**：`/admin/plans/:planId/liabilities`
- **功能**：配置方案的责任和保额选项
  - 查看方案的责任配置
  - 添加/编辑/删除责任配置
  - 设置保额选项、默认值等

#### 2.6 费率配置 (RateList)
- **路径**：`/admin/rates`
- **功能**：管理保费费率
  - 查看费率列表
  - 新增/编辑/删除费率
  - 按产品、责任、职业类别筛选

#### 2.7 接口配置 (ApiConfigList)
- **路径**：`/admin/api-configs`
- **功能**：管理保司API接口配置
  - 查看接口配置列表
  - 新增/编辑/删除接口配置
  - 配置API端点、字段映射等

### 3. 配置导入 (ConfigImport)
- **路径**：`/admin/config-import`
- **功能**：批量导入配置
  - 上传配置文件
  - 预览导入数据
  - 执行导入操作

### 4. 业务管理

#### 4.1 投保单管理 (ApplicationsList) ⭐ 新增
- **路径**：`/admin/applications`
- **功能**：管理投保单
  - 查看投保单列表
  - 查看投保单详情
  - 按状态、日期、企业筛选
  - 支持搜索投保单号、企业名称

#### 4.2 企业客户管理 (CompanyList) ⭐ 新增
- **路径**：`/admin/companies`
- **功能**：管理企业客户
  - 查看企业列表
  - 新增/编辑/删除企业
  - 搜索企业名称、信用代码、联系人
  - 查看企业详细信息

### 5. 拦截规则管理 (InterceptionRulesList) ⭐ 新增
- **路径**：`/admin/interception-rules`
- **功能**：管理承保拦截规则
  - 查看拦截规则列表
  - 新增/编辑/删除拦截规则
  - 配置规则类型、条件、动作
  - 设置规则优先级

### 6. 数据统计 (Statistics) ⭐ 新增
- **路径**：`/admin/statistics`
- **功能**：数据统计和报表
  - 总投保单数
  - 企业客户数
  - 总保费
  - 总被保人数
  - 月度投保趋势图表
  - 产品统计表格

### 7. 系统日志 (SystemLogs) ⭐ 新增
- **路径**：`/admin/system-logs`
- **功能**：查看系统操作日志
  - 查看操作日志、错误日志、登录日志
  - 按类型、模块、日期筛选
  - 搜索日志内容
  - 导出日志

## 二、页面功能特点

### 通用功能
- ✅ 列表展示（表格）
- ✅ 搜索和筛选
- ✅ 分页
- ✅ 新增/编辑/删除
- ✅ 详情查看
- ✅ 状态管理（启用/禁用）

### 特殊功能
- 📊 **数据统计**：图表展示、数据汇总
- 📝 **系统日志**：日志查询、导出
- 🔍 **投保单管理**：多条件筛选、状态跟踪
- 👥 **企业客户管理**：客户信息管理
- 🚫 **拦截规则**：规则配置、优先级管理

## 四、技术实现

### 使用的技术栈
- **React 18** + **TypeScript**
- **Ant Design 5** - UI组件库
- **React Router 6** - 路由管理
- **@ant-design/plots** - 图表库
- **dayjs** - 日期处理

### 页面组件结构
```
admin/src/pages/
├── Dashboard.tsx              # 仪表盘
├── InsuranceCompanyList.tsx    # 保司管理
├── LiabilityList.tsx           # 责任管理
├── ProductList.tsx             # 产品管理
├── PlanList.tsx               # 方案管理
├── PlanLiabilityConfig.tsx     # 方案责任配置
├── RateList.tsx               # 费率配置
├── ApiConfigList.tsx           # 接口配置
├── ConfigImport.tsx            # 配置导入
├── ApplicationsList.tsx        # 投保单管理 ⭐ 新增
```

## 六、使用说明

### 启动后台管理系统
```bash
cd admin
npm install
npm run dev
```

### 访问地址
- 开发环境：http://localhost:3001/admin
- 登录页面：http://localhost:3001/login

### 注意事项
- 所有页面目前使用Mock数据
- 需要连接后端API才能正常使用
- 部分功能（如导出）还在开发中
