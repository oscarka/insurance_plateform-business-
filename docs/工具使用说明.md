# 数据库结构图查看工具使用说明

## 推荐工具（按推荐顺序）

### 1. dbdiagram.io（最推荐）⭐

**优点**：
- 在线工具，无需安装
- 支持中文，界面友好
- 自动生成美观的ER图
- 支持导出为图片或PDF

**使用方法**：
1. 访问：https://dbdiagram.io
2. 点击 "Import" 或 "New Project"
3. 选择 "Import from DBML"
4. 打开项目中的 `数据库结构图.dbml` 文件
5. 或者直接复制文件内容粘贴进去
6. 即可看到可视化的数据库结构图

**导出**：
- 可以导出为 PNG、PDF、SQL 等格式

---

### 2. Mermaid Live Editor

**优点**：
- 在线工具，无需安装
- 支持 Mermaid 语法
- 可以导出为 SVG、PNG

**使用方法**：
1. 访问：https://mermaid.live
2. 打开项目中的 `数据库结构图.md` 文件
3. 复制 Mermaid 代码块（```mermaid 到 ```之间的内容）
4. 粘贴到 Mermaid Live Editor
5. 即可看到图形

**注意**：
- 如果图表太大，可能需要分块查看

---

### 3. Draw.io (diagrams.net)

**优点**：
- 免费，功能强大
- 支持在线和桌面版
- 可以导入多种格式

**使用方法**：
1. 访问：https://app.diagrams.net 或下载桌面版
2. 选择 "Create New Diagram"
3. 可以选择导入 Mermaid 或手动绘制
4. 或者使用 "Insert" → "Advanced" → "Mermaid" 插入 Mermaid 代码

**桌面版下载**：
- https://github.com/jgraph/drawio-desktop/releases

---

### 4. MySQL Workbench

**优点**：
- 专业的数据库设计工具
- 可以生成ER图
- 支持反向工程

**使用方法**：
1. 下载安装 MySQL Workbench
2. 创建新的 EER Model
3. 可以手动创建表，或者导入 SQL 脚本
4. 自动生成ER图

**下载地址**：
- https://dev.mysql.com/downloads/workbench/

---

### 5. DBeaver

**优点**：
- 免费的数据库管理工具
- 可以连接数据库后自动生成ER图
- 支持多种数据库

**使用方法**：
1. 下载安装 DBeaver
2. 连接数据库
3. 右键数据库 → "View Diagram"
4. 自动生成ER图

**下载地址**：
- https://dbeaver.io/download/

---

### 6. VS Code 插件（推荐给开发者）

**优点**：
- 在编辑器中直接查看
- 无需切换工具

**推荐插件**：
1. **Mermaid Preview** - 预览 Mermaid 图表
2. **Markdown Preview Mermaid Support** - Markdown 中的 Mermaid 支持
3. **DBML Preview** - 预览 DBML 文件

**使用方法**：
1. 在 VS Code 中安装插件
2. 打开 `数据库结构图.md` 或 `数据库结构图.dbml`
3. 使用预览功能查看图表

---

## 快速开始（最简单的方法）

### 方法一：dbdiagram.io（推荐）

1. **打开浏览器**，访问：https://dbdiagram.io
2. **点击右上角** "Try it free" 或直接使用
3. **点击左侧菜单** "Import" 按钮
4. **选择** "Import from DBML"
5. **打开项目中的** `数据库结构图.dbml` 文件
6. **完成！** 即可看到完整的数据库结构图

### 方法二：Mermaid Live Editor

1. **打开浏览器**，访问：https://mermaid.live
2. **打开项目中的** `数据库结构图.md` 文件
3. **找到第一个** ```mermaid 代码块
4. **复制** 代码块内容（不包括 ```mermaid 和 ```）
5. **粘贴** 到 Mermaid Live Editor
6. **完成！** 即可看到图表

## 常见问题

### Q: dbdiagram.io 需要注册吗？
A: 不需要，可以直接使用，但注册后可以保存项目。

### Q: 图表太大看不清怎么办？
A: 
- dbdiagram.io 支持缩放和拖拽
- 可以导出为高清图片
- 或者分模块查看

### Q: 可以编辑图表吗？
A: 
- dbdiagram.io 支持在线编辑
- Draw.io 功能最强大，支持各种编辑

### Q: 可以导出为其他格式吗？
A: 
- dbdiagram.io 支持导出 PNG、PDF、SQL
- Mermaid Live Editor 支持导出 SVG、PNG
- Draw.io 支持导出多种格式
